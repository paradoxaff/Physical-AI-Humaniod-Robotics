# API Contracts

This directory contains the API contracts for the Physical AI & Humanoid Robotics textbook project.

## Available Contracts

- `textbook-api.yaml` - OpenAPI specification for the textbook API, including:
  - Lesson and module content retrieval
  - RAG (Retrieval Augmented Generation) query endpoints
  - User profile and progress management
  - Hardware configuration details

## API Design Principles

- RESTful design where appropriate
- JSON request/response payloads
- Standard HTTP status codes
- Consistent error response format
- Proper authentication and authorization patterns

## Versioning

APIs are versioned to ensure backward compatibility. The current version is 1.0.0.
